# **NodeServer**


**Node 모듈**을 사용해 **Server**를 구성 후 아두이노, 데이터베이스, 웹&앱 과 연결

## 사용된 module

>**Node : 10.16.0(x64)** [다운로드](https://nodejs.org/ko/)

>**npm : 6.9.0**
>>npm 버전확인
```
npm -v
```
>npm package 다운로드
```
npm install
```
>npm 업데이트
>>>리눅스
```
sudo npm install -g npm
```
>>>윈도우
```
npm install -g  npm
```
